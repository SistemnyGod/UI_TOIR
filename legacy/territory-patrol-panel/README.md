# Territory patrol panel

Статус: legacy prototype.

Это старый статический UI-прототип без backend-интеграции. Активный frontend находится в `apps/web`.

Назначение каталога:

- историческая визуальная reference-точка;
- проверка, что полезные UI-решения уже перенесены в React-приложение;
- временное хранение до окончательного удаления или архивации.

Этот каталог не должен участвовать в CI, production build и новой feature-разработке.
