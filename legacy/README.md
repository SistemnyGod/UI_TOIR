# Legacy

Каталог содержит материалы, которые больше не являются активной production-частью проекта.

Правила:

- CI не собирает legacy-код.
- Новая разработка ведется в `apps`, `libs`, `infra`, `tools`, `tests` и `docs`.
- Перед удалением legacy-файлов нужно убедиться, что полезные сценарии перенесены в активный код или документацию.
