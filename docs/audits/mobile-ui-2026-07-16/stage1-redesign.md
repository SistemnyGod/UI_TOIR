# Этап 1 — редизайн интерфейса «Обход»

## Изменённые поверхности

1. Список заявок: удалены обучение, четыре статистические карточки, большая кнопка обновления и повторные подсказки; обновление перенесено в pull-to-refresh.
2. Проверка заявки: оставлена компактная карточка и одно действие по состоянию; возврат заявки перенесён в меню.
3. Активный обход: одно действие — старт, NFC, проверка отчёта, повтор отправки или переход к заявкам; QR, пауза и передача диспетчеру перенесены в меню.
4. Точка: три выбора состояния; четыре медиа-кнопки объединены в «Добавить вложение»; «Отложить» перенесено в меню; сохранение стало действием «Сохранить и продолжить».
5. Проверка отчёта: при проблемах главное действие открывает первую проблемную точку; отправка появляется только для готового отчёта.

## Перенос действий

| Действие | Новое расположение |
|---|---|
| Обновить заявки | Pull-to-refresh списка |
| Вернуть заявку | Меню «Действия с заявкой» |
| QR-резерв | Меню «Действия с обходом» |
| Приостановить | Меню «Действия с обходом» |
| Передать диспетчеру | Меню «Действия с обходом» |
| Фото/видео: камера/галерея | Action sheet «Добавить вложение» |
| Отложить точку | Меню «Действия с точкой» |
| Все точки / очередь | Компактные вторичные ссылки |
